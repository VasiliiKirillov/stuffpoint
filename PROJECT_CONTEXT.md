# ai.stuff — Project Context

## 1) Что это за проект
- Название: `ai.stuff`
- Тип: статический лендинг + страницы направлений специалистов
- Стек: чистый `HTML + CSS + JS` (без фреймворков)
- Репозиторий: `git@github.com:VasiliiKirillov/stuffpoint.git`
- Ветка деплоя: `main`
- Хостинг: GitHub Pages
- Кастомный домен: `ai-stuff.ru`

## 2) Главная идея позиционирования
`ai.stuff` — кадровое агентство нового типа:
- подбирает IT-специалистов,
- внедряет AI-ассистентов и AI-операции,
- может строить гибридную модель «люди + AI».

Ключевой hero-текст на текущий момент:
- `Подбор IT специалистов и AI-агентов для вашего бизнеса.`
